# ToDo-List

ToDo list is a web application to save Your daily tasks in order not to miss anything.

### Features
* Add new task
* Mark task as completed / uncompleted
* Edit task
* Delete task

### Technologies used:
* HTML
* CSS
* JavaScript
* Node.js
* Mongodb/mongoose
* express.js
* ejs
